Terms and Definitions

DOCTYPE statement
It is an "information" to the browser about what document type to expect.
How to use (ol), (ul), and (dl)
(ol) is used when the list is ordered (numbers or other notation) and the ul element is used when the list is unordered (usually bullets). Definition lists (dl) are used to group terms with their definitions, like with this list.
Text Editor vs Word Processor
A text editor is a system that allows users to edit and store text, while programming code. A Word Processor allows users to write and edit documents, but does not contain the same code formatting as with text editors
Client Server Model
A Client Server Model allows a client, like a PC, to upload data to the server over a local network or the Internet. For example, a laptop using the internet to access a web server and the html data there.
FTP
FTP is File Transfer Protocol. This is a network protocol for transmitting files between computers over TCP/IP using control and data connections between the client and the server..
HTTP
HTTP stands for Hypertext Transfer Protocol. It is the set of rules for transferring files (PDF, PNG, JPG, DOC) over the web. Users access HTTP automatically on web servers such as chrome.
TCP/IP
TCP/IP stands for Transmission Control Protocol/Internet Protocol and refers to data and communication on a private network.
